Calculator

This is very basic application I worked on. It's a single view application. I created this application in my android development course. This was my first mobile application I have ever created. This application contains a text view with lots of buttons just like a regular calculator.

User can do many mathematical calculation in this application such as addition, subtraction, multiplication, division. Other than these, user can also calculate square of any number and can calculate percentage. While developing my first android application I learned lot of things about development and designing of applications. I came to know how application adjust their sizes according to the size of screen of smartphone. I worked with constraints while making this Calculator app. As mentioned, this is single screen application. Therefore, it only have one screenshot.
